### 回溯算法

解决问题

- 组合问题：N个数按一定规则找出k个数的集合
- 切割：一个字符按一定规则有几种切割方式
- 子集问题：一个N个数的集合里有多少符合条件的子集
- 排列：N个数按一定规则全排列，有几种排列方式
- 棋盘：N皇后，解数独等等
- 

回溯法模板：

```c++
void backtracking(参数) {
	if (终止条件) { 收集结果; return; }
	for(集合的元素集)	{ 处理节点;递归函数;回溯操作(撤销节点的处理情况);}
    { return;}
}
```

步骤：

1. 递归函数参数返回值
2. 确定终止条件
3. 单层递归逻辑
